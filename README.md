ESLint Library bundle for us.

## Rules

### `@jamashita/eslint-config`

### Append in `.eslintrc`

```json
{
  "extends": [
    "@jamashita"
  ]
}
```

### `@jamashita/eslint-config-jest`

Jest config. Jest required.

### Append in `.eslintrc`

```json
{
  "extends": [
    "@jamashita/jest"
  ]
}
```

### `@jamashita/eslint-config-react`

React config. React required.

### Append in `.eslintrc`

```json
{
  "extends": [
    "@jamashita/react"
  ]
}
```
